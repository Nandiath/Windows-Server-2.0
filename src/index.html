O Que Você Aprenderá Conosco?

Este programa de formação em Windows Server Hybrid Administrator foi desenvolvido para capacitar os participantes com as habilidades necessárias para administrar, configurar e otimizar ambientes de servidores híbridos utilizando tecnologias Microsoft.

6H

Introdução à Computação em Nuvem

Fundamentos de cloud computing, modelos de serviço (IaaS, PaaS, SaaS), Windows Server Híbrido e arquiteturas de infraestrutura híbrida.

12H

Gerenciamento de Servidores Windows Server

Instalação e configuração do Windows Server 2019/2022, Active Directory híbrido, Azure AD e integração com Azure.

10H

Gerenciamento de Rede e Armazenamento

Configuração de DHCP, DNS, VPN, armazenamento local/Azure, backup e recuperação de desastres em ambientes híbridos.

8H

Provisionamento de Desktops Virtuais

Implementação do Azure Virtual Desktop, publicação de aplicativos, gestão de usuários e segurança de ambientes virtuais.

3H

Domine a Administração Híbrida

Aprenda a gerenciar ambientes locais e na nuvem com Windows Server Hybrid. Torne-se um especialista em infraestruturas híbridas.
